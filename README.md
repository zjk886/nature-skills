# Nature-Style Academic Polishing Skill

一个基于 Nature 正刊论文写作风格分析构建的学术润色 skill，可将中文学术草稿转化为符合 Nature 期刊标准的英文学术文本。

## 数据来源

从本目录下 18 篇 Nature 正刊论文（s41586 系列，2026 年发表）中抽样精读 5 篇，覆盖天体化学、电化学、光学显示、分子生物学、植物学等领域，提取以下风格特征：

- 摘要结构与引言漏斗式推进
- 句型模板与主动语态偏好
- 模糊语（hedging）的三级校准
- 弱动词替换策略
- 段落"论点-证据-解释-过渡"模式

## 核心能力

| 能力 | 说明 |
|------|------|
| 长句拆分 | 中文 >40 字符或多逗号句子自动拆为 2-3 个英文短句，每句 ≤30 词 |
| 词汇升级 | 弱动词/模糊形容词替换为精确学术用语 |
| 模糊语校准 | 按证据强度匹配用词（demonstrate → suggest → may reflect） |
| 结构重组 | 摘要/引言按 Nature 标准模板对齐 |
| 即贴即用 | 输出为纯文本，可直接复制粘贴到论文草稿中 |

## 使用方法

让 Claude 执行以下任意一种指令即可触发：

- "用 nature-polishing 润色这段文字"
- "把这段话改成 Nature 风格"
- "polish this in Nature style"

## 适用场景

- 中文学术草稿的英文翻译与润色
- 已有英文文本的 Nature 风格改写
- 摘要、引言、结论段落的结构优化
- 论文返修时的语言升级

## 不适用场景

- 完整论文从头撰写（应参考具体的期刊 author guidelines）
- 非学术类写作（科普、博客、新闻报道等）
- 需要保留原文特定术语或作者个人风格的情况
